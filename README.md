# ElasticSearch Spring Boot CRUD 實作

### Elastic Search Docker 安裝: 
``` bash
進入 https://github.com/ShaoweiTeng-lab/elasticSearch_Docker 專案
依據說明啟動 ELK
```
## 實作項目:

	•Spring Boot整合 Elasticsearch 
	•實作全文搜尋 API（match / multi_match）
	•分頁與排序功能（Pageable）
	•搜尋高亮（highlight）
	•例外處理與統一回傳格式設計
